Function Documentation#

template<typename T>
auto einsums::blas::getrf(eint, eint, T*, eint, eint*) -> eint#

Computes the LU factorization of a general M-by-N matrix A using partial pivoting with row einterchanges.

The factorization has the form A = P * L * U where P is a permutation matri, L is lower triangular with unit diagonal elements (lower trapezoidal if m > n) and U is upper triangular (upper trapezoidal if m < n).
