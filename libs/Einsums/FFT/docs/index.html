FFT#

This module contains wrappers for performing the fast Fourier transform.

See the API reference of this module for more details.

Public API#

void einsums::fft::fft(Tensor<float, 1> const &a, Tensor<std::complex<float>, 2> *result)#
void einsums::fft::fft(Tensor<double, 1> const &a, Tensor<std::complex<double>, 2> *result)#
void einsums::fft::fft(Tensor<std::complex<float>, 1> const &a, Tensor<std::complex<float>, 2> *result)#
void einsums::fft::fft(Tensor<std::complex<double>, 1> const &a, Tensor<std::complex<double>, 2> *result)#

Performs the fast Fourier transform on the input tensor using the linked FFT library.

Parameters:

  • a[in] – The input tensor.

  • result[out] – The output tensor.

void einsums::fft::ifft(Tensor<std::complex<float>, 1> const &a, Tensor<float, 2> *result)#
void einsums::fft::ifft(Tensor<std::complex<double>, 1> const &a, Tensor<double, 2> *result)#
void einsums::fft::ifft(Tensor<std::complex<float>, 1> const &a, Tensor<std::complex<float>, 2> *result)#
void einsums::fft::ifft(Tensor<std::complex<double>, 1> const &a, Tensor<std::complex<double>, 2> *result)#

Performs the inverse fast Fourier transform on the input tensor using the linked FFT library.

Parameters:

  • a[in] – The input tensor.

  • result[out] – The output tensor.

Tensor<double, 1> fftfreq(int n, double d)#

Gets the frequency corresponding to each position in the frequency tensor.

Parameters:

  • n – The number of positions in the tensor.

  • d – The scale factor for the frequency.

Returns:

A tensor that contains the frequency at each position in a frequency tensor of the same size.
