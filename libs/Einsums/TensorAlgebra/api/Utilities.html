template<size_t Rank, typename ...Args>
auto order_indices(std::tuple<Args...> const &combination, std::array<size_t, Rank> const &order)#

Swap around the indices to the desired order. For instance, if we have a list of indices {a,b,c} and we want to reverse it, we would give the order {2, 1, 0}. There is no requirement that all the orders be different, so if we were to pass {1, 1, 0} instead, we would get {b,b,a}.

Parameters:

  • combination – The indices to reorder.

  • order – The array containing the new order.
