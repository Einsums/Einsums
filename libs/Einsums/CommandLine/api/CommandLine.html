Parses command-line arguments storing their presence into previously registered Opt/Flag/OpenEnum option.

Parameters:

  • args – command-line arguments converted to a std::vector<std::string>

  • programName – the program name to display in help printing

  • version – the program version to display in version printing

  • unknown_args – arguments not understood by our parser are placed here

Returns:

if ParseResult.ok is true then parsing completed successfully

Parses command-line arguments storing their presence into previously registered Opt/Flag/OpenEnum option.
