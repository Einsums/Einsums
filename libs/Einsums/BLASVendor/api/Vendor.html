int_t ssyev(char job, char uplo, int_t n, float *a, int_t lda, float *w, float *work, int_t lwork)#

Performs symmetric matrix diagonalization.

Added in version 1.0.0.

Parameters:

  • job – [in] Whether to compute the eigenvectors. Case insensitive. Can be either ‘n’ or ‘v’.

  • uplo – [in] Whether the data is stored in the upper or lower triangle of the input. Case insensitive. Can be either ‘u’ or ‘l’.

  • n – [in] The number of rows and columns of the matrix.

  • a – [inout] The matrix to factor. On exit, its data is overwritten. If the vectors are computed, they are placed in the columns of the matrix.

  • lda – [in] The leading dimension of the matrix.

  • w – [out] The output for the eigenvalues.

  • work – [inout] A work array. If lwork is -1, then the optimal work array size is placed in the first element of this array.

  • lwork – [in] The size of the work array. If it is -1, then a workspace query is performed, and the optimal workspace size is put in the first element of work.

Returns:

0 on success. If positive, then the eigenvalue algorithm did not converge. If negative, then one of the parameters was given a bad value. The absolute value indicates which parameter it was.

  • rwork – [inout] A work array for real data. Must be at least \(3n - 2\) elements.

Returns:
