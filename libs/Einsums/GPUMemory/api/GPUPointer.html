template<typename T>
struct GPUPointer#
#include <GPUPointer.hpp>

A pointer to memory on the GPU.

This is used to wrap up certain operations. Unfortunately, it can not be used as a normal fancy pointer since the addressof operation does not make sense for its references, which actually handle the data transfers. However, for the majority of use cases, it can be used as a fancy pointer.

Added in version 1.1.0.

Template Parameters:

The – host-side type held by the pointer.
