template<typename D>
bool IsFunctionTensorV = requires {requires detail::test_application<D>(std::make_index_sequence<std::remove_cvref_t<D>::Rank>());requiresIsRankTensorV<D>;}#

Checks to see if the tensor is a function tensor.

More specifically, checks to see if the tensor can be indexed using function call syntax, and it provides a function called subscript. The subscript function should not do any bounds checking or negative index checking, whereas the function call operator should.

Template Parameters:

D – The tensor type to check.

template<typename D>
bool IsFastSubscriptableV = requires {requires detail::test_fastsubscript<D>(std::make_index_sequence<std::remove_cvref_t<D>::Rank>());requiresIsRankTensorV<D>;}#

Checks to see if the tensor provides the subscript function.

The subscript function should not do any bounds checking, nor should it accept negative indices as valid, unless negative indices have some sort of meaning outside of index wrapping. These checks add up when iterating over a tensor with millions of entries, so being able to skip them when bounds compliance is guaranteed can speed up code.
