class Plan#
#include <Plan.hpp>

A plan encodes the execution of a tensor transposition.

It stores the loop order and parallelizes each loop.

Public Functions

inline Plan()#
Plan(std::vector<int> loopOrder, std::vector<int> numThreadsAtLoop)#

Construct a plan. Initialize the loop order and the number of threads for each level of loop.

Parameters:

  • loopOrder – Holds the indices in the order they are to be updated.

  • numThreadsAtLoop – Contains the number of threads for each level of loop. Used to find the number of tasks as well.

std::vector<int> loopOrder_#

Holds the indices in the order that they will be used.

For example, if \( B_{1,0,2} \gets A_{0,1,2}\). loopOrder_ = {1,0,2} denotes that B is traversed in a linear fashion.
