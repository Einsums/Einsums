enum class RuntimeState : std::int8_t#

Holds the possible states for the runtime.

Added in version 1.0.0.

Values:

enumerator Invalid#

The state is invalid.

enumerator Initialized#

The runtime has been initialized.

enumerator PreStartup#

The runtime is running the pre-startup functions.

enumerator Startup#

The runtime is running the startup functions.

enumerator PreMain#

The runtime is preparing to run the main function.

enumerator Starting#

The runtime is starting the main function.

enumerator Running#

The main function is running.

enumerator PreShutdown#

The pre-shutdown functions are running.

enumerator Shutdown#

The shutdown functions are running.

enumerator Stopping#

The runtime is stopping.

enumerator Terminating#

The runtime is terminating.

enumerator Stopped#

The runtime has stopped.

enumerator LastValidState#

Indicates the last valid state. Anything past this is considered invalid.

bool is_running()#

Test whether the runtime system is currently running.

This function returns whether the runtime system is currently running or not, e.g. whether the current state of the runtime system is einsums::RuntimeState::Running

Note

This function needs to be executed on a pika-thread. It will return false otherwise.

Added in version 1.0.0.
