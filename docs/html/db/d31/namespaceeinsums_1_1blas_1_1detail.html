Detailed Description

Performs matrix multiplication for general square matices of type double.

Performs matrix vector multiplication.

Performs symmetric matrix diagonalization.

Computes all eigenvalues and, optionally, eigenvectors of a Hermitian matrix.

Computes the solution to system of linear equations A * x = B for general matrices.

Return the value of the 1-norm, Frobenius norm, infinity-norm, or the largest absolute value of any element of a general rectangular matrix

Computes the singular value decomposition of a general rectangular matrix using a divide and conquer method.

Computes a QR factorizaton (Useful for orthonormalizing matrices)

Returns the orthogonal/unitary matrix Q from the output of dgeqrf
