1 # Einsums in C++
2 
3 Provides compile-time contraction pattern analysis to determine optimal operation to perform.
4 
5 ## Requirements
6 A C++ compiler with C++17 support.
7 
8 The following libraries are required to build EinsumsInCpp:
9 
10 * BLAS and LAPACK.
11 
12 On my personal development machine, I use MKL for the above requirements. On GitHub Actions, stock BLAS, LAPACK, and FFTW3 are used.
13 
14 Optional requirements:
15 
16 * A Fast Fourier Transform library, either FFTW3 or DFT from MKL.
17 * For call stack backtracing, refer to the requirements listed [here](https://github.com/bombela/backward-cpp).

20 This will optimize at compile-time to a BLAS dgemm call.
21 ```C++
22 #include "einsums/TensorAlgebra.hpp"
23 
24 using einsums; // Provides Tensor and create_random_tensor
25 using einsums::TensorAlgebra; // Provides einsum and Indices

28 Tensor<2> A = create_random_tensor("A", 7, 7);
